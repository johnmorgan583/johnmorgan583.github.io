A couple takeaways:

Attrition is pretty imbalanced, which may be an issue when predicting it.

A few variables have the same value for the whole column and should be removed.

Removing variables with no variability.

cleandata<-data %>% 
  select(-EmployeeCount,-Over18,-StandardHours)

Correlation plot of numeric variables only.

nums <- cleandata %>% 
  mutate(Attrit.Num=recode(Attrition,"No"=0,"Yes"=1)) %>% 
  select(is.numeric)

corrplot::corrplot(cor(nums),tl.cex=.7)

No clear driver, but a few small negative correlations with attrition.

Correlation funnel focused on our outcome of interest, this time including categorical variables.

bin<-correlationfunnel::binarize(cleandata,n_bins=5,thresh_infreq = 0.01, name_infreq = "OTHER", one_hot = TRUE)
cortable<-bin %>% correlationfunnel::correlate(Attrition__Yes)
correlationfunnel::plot_correlation_funnel(cortable)

Overtime seems to have the strongest relationship with attrition but is still fairly low at around .25.

Initial Predictive Model

Setting up our train/test split using tidymodels.

set.seed(777)

splitter<-initial_split(cleandata,
                        prop=7/10) #70/30 split

train <- training(splitter)
test <- testing(splitter)

Setting up our recipe and model. I chose random forest due to it usually having good performance with minimal tuning.

att.recipe <- recipe(Attrition~.,data=train) 

rf.model <- rand_forest() %>% 
  set_mode("classification") %>% 
  set_engine(engine="randomForest")

ml.workflow <- workflow() %>% 
  add_recipe(att.recipe) %>% 
  add_model(rf.model)

Let’s try this again with random oversampling!

Predictive Model Using Random Oversampling

att.recipe.rose <- recipe(Attrition~.,data=train) %>% 
  themis::step_rose(Attrition) #Adding the random oversampling to our recipe.

rf.model <- rand_forest() %>% 
  set_mode("classification") %>% 
  set_engine(engine="randomForest")

ml.workflow.rose <- workflow() %>% 
  add_recipe(att.recipe.rose) %>% 
  add_model(rf.model)

Using the Model

For fun, using demographic info about me, let’s predict my likelihood of attrition.

predictor <- test.model.rose %>% extract_workflow() #extracting our model for predictions

itsa.me<-read_excel("john.xlsx")

predict(predictor,
        new_data=itsa.me,
        type="prob")
## # A tibble: 1 × 2
##   .pred_No .pred_Yes
##      <dbl>     <dbl>

Looks like I’ll be “randomly” selected for a stay interview in the near future!
